# 🚖 DriveBy Taxi App

DriveBy Taxi App is a responsive front-end taxi ride booking application developed using **HTML, CSS, and JavaScript**. The project provides a clean and user-friendly interface that allows customers to view booking statistics, book taxi rides, and navigate through different sections of the website. It serves as a starter template for a taxi booking platform and demonstrates the use of basic client-side form validation and multi-page website navigation.

---

## 📌 Features

- Responsive and user-friendly interface
- Home dashboard displaying:
  - Total Cabs
  - Total Bookings
  - Completed Rides
  - Cancelled Rides
  - Recent Booking Details
- Book Cab page with booking form
- Basic JavaScript form validation
- Login page with Email and Password
- About Us page
- Contact Us page
- Common navigation bar across all pages
- Footer with quick links and contact information

---

## 🏠 Pages

### Home
Displays a dashboard containing:
- Total Cabs
- Total Bookings
- Completed Bookings
- Cancelled Bookings
- Recent Bookings Table

### Book Cab
Allows users to book a taxi by entering:
- Customer Name
- Phone Number
- Pickup Location
- Drop Location
- Available Cab Selection

The booking form includes basic validation to ensure required fields are filled before submission.

### About Us
Provides information about the DriveBy Taxi App, its mission, and the services offered.

### Contact Us
Displays company contact information and allows users to submit inquiries.

### Login
A simple login page where users can sign in using:
- Email Address
- Password
